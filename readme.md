## FuckUnity.js
### Remove the Unity engine's obstruction to game program launch on the web platform
### 删除Web平台上的Unity引擎对游戏程序的启动阻碍

<hr>

## How we did it
### At present, we have forged the Webview's UserAgent to meet the requirements of some web games for a PC, and then trick UnityLoader.js into thinking the current device is a desktop, allowing it to run.
### 目前，我们伪造了Webview的UserAgent达到部分网页游戏需要PC端的需求，然后欺骗UnityLoader.js认为当前设备是桌面设备，使其驱动。
